# Layer 4 Recommendation Catalog

## Purpose

This catalog defines the first AI recommendation types Beeooking should support. Each recommendation should include an explanation, confidence score, source data summary, permission-aware audience, and measurable outcome.

## AI Coach Recommendations

### Clinic Recommendation

Audience:

- Coach
- Parent
- Player

Inputs:

- Player profile
- Program eligibility
- Attendance history
- Skill assessments
- Development goals
- Coach notes

Recommended action:

- Register for a specific clinic, camp, course, or training group.

Outcome measures:

- Recommendation accepted
- Program registration created
- Payment completed
- Attendance recorded
- Goal progress updated

### Coach Recommendation

Audience:

- Club admin
- Coach
- Parent
- Player

Inputs:

- Player goals
- Skill gaps
- Coach sports
- Coach availability
- Booking history
- Program history

Recommended action:

- Book a session with a specific coach.

Outcome measures:

- Booking created
- Session attended
- Follow-up note created
- Player/parent feedback submitted

### Opponent Recommendation

Audience:

- Coach
- Player
- Parent

Inputs:

- Match history
- Ladder rank
- Standings
- Skill level
- Age/eligibility rules
- Recent opponent history

Recommended action:

- Challenge or schedule a match with a suggested opponent.

Outcome measures:

- Challenge created
- Match scheduled
- Match completed
- Result submitted

### Training Plan Recommendation

Audience:

- Coach first
- Parent/player after approval

Inputs:

- Goals
- Skill assessments
- Attendance
- Coach notes
- Match outcomes
- Program participation

Recommended action:

- Follow a specific development plan with goals, programs, practice focus, and competition targets.

Outcome measures:

- Goal created or updated
- Program registration
- Attendance consistency
- Assessment improvement
- Match performance change

## AI Club Manager Recommendations

### Membership Renewal Recommendation

Audience:

- Club admin

Inputs:

- Membership status
- Renewal date
- Attendance
- Booking activity
- Payment history
- Communication history

Recommended action:

- Contact a member or family before renewal risk increases.

Outcome measures:

- Message sent
- Renewal completed
- Membership retained
- Payment completed

### Retention Campaign Recommendation

Audience:

- Club admin

Inputs:

- Activity decline
- Missed attendance
- Booking drop-off
- Program inactivity
- Failed payments
- Lapsed memberships

Recommended action:

- Create a targeted retention campaign for a member segment.

Outcome measures:

- Campaign draft created
- Message sent
- Booking or registration created
- Membership renewed

### Program Opportunity Recommendation

Audience:

- Club admin

Inputs:

- Court utilization
- Program demand
- Waitlists
- Skill assessment clusters
- Membership mix
- Coach availability

Recommended action:

- Launch a specific clinic, camp, course, or league.

Outcome measures:

- Program created
- Registrations created
- Revenue generated
- Attendance recorded
- Waitlist reduced

## AI Player Development Recommendations

### Development Area Recommendation

Audience:

- Coach
- Parent/player after approval

Inputs:

- Skill assessments
- Coach notes
- Match results
- Attendance
- Goals

Recommended action:

- Focus on a specific development area.

Outcome measures:

- Goal created
- Coach note follow-up
- Assessment score improvement
- Report card inclusion

### Next Program Recommendation

Audience:

- Coach
- Parent
- Player

Inputs:

- Current development stage
- Program history
- Assessment scores
- Goals
- Attendance
- Coach notes

Recommended action:

- Join the next best program or training group.

Outcome measures:

- Registration created
- Attendance recorded
- Goal progress updated

### Competitive Pathway Recommendation

Audience:

- Coach
- Parent/player after approval

Inputs:

- Match history
- Ladder activity
- Tournament participation
- Skill assessments
- Attendance
- Goals

Recommended action:

- Move toward a ladder, league, round robin, tournament, or external competition path.

Outcome measures:

- Competition registration
- Challenge created
- Match completed
- Standings/ranking movement
